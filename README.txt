Bear Travel Planner 1.0 資料穩定正式版

重點：
1. 從此版本開始固定使用同一個資料庫 key：bearTravelPlanner。
2. 更新網頁檔案不會因為版本號改變而看不到舊資料。
3. 第一次打開會自動尋找舊版資料（v10~v21、1.0 舊版等），並自動搬移到正式資料庫。
4. 搬移前會留下 migration backup，並保留最新自動備份。
5. 工具頁仍可匯出/匯入完整備份，正式使用前建議先匯出一次。

使用方式：
- 解壓縮後將所有檔案上傳/覆蓋原本 GitHub Pages 專案。
- 網址保持同一個，資料會留在同一個網域的瀏覽器中。
- 換手機或清除 Safari 網站資料前，請務必先按「一鍵完整備份」。
